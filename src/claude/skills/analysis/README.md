# Analysis Skills - 分析评估

需求分析和评估功能，提供科学的优先级评估和度量系统。

## 技能列表

### req-priority
优先级评估 - 5维度科学评估（业务价值、紧急程度、技术可行性、依赖关系、ROI）。

### req-metrics
度量系统 - 4维度16指标，追踪项目健康状态。

## 评估维度

1. **业务价值** (40%) - 对业务的贡献程度
2. **紧急程度** (30%) - 时间敏感性
3. **技术可行性** (15%) - 实现难度
4. **依赖关系** (10%) - 前置依赖
5. **ROI** (5%) - 投资回报率
