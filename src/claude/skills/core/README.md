# Core Skills - 核心需求管理

核心需求管理功能，提供统一入口和深度分析能力。

## 技能列表

### req-manager

需求管理统一入口 - 智能路由到相应的处理流程，简化用户操作。

### req-brainstorm

深度需求分析模式 - 结合头脑风暴与无情审查，确保需求分析的深度和完整性。

### setup

初始化配置 - 一次性设置，配置问题追踪系统、标签体系和文档存储位置。

## 使用流程

1. 运行 `/setup` 完成初始化
2. 使用 `/req` 添加需求（自动路由）
3. 对于复杂需求，使用 `/req-brainstorm` 深度分析
