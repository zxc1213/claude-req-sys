# Quality Skills - 质量保证

质量保证和验证功能，确保需求交付质量。

## 技能列表

### quality-gates
质量门禁系统 - 4阶段自动质量检查（创建、分析、实现、完成）。

### verification-checklist
验证清单 - 系统化质量验证检查项，覆盖关键阶段。

### test-plan-generator
测试计划生成器 - 自动生成测试计划和测试用例。

## 质量流程

```
需求创建 → 质量门禁 #1 → 需求分析 → 质量门禁 #2
    ↓
实现开发 → 质量门禁 #3 → 验收测试 → 质量门禁 #4 → 完成
```
